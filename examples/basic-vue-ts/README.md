# basic-vue-ts

This is an example of using `vue-material-adapter` with a typescript project generated with `create-vue`.

## Quick start

```bash
npm install
npm run dev
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```
